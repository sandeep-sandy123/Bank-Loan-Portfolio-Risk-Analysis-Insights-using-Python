# Bank-Loan-Portfolio-Risk-Analysis-Insights-using-Python
This project focuses on analyzing historical bank loan data to identify risk patterns and predict loan default probability using Python.

1. 📌 Project Overview

This project analyzes historical bank loan data to identify risk patterns and predict the likelihood of loan default.
The objective is to simulate a real-world credit risk assessment system that helps financial institutions make data-driven lending decisions and minimize financial losses.
The project follows a complete data science workflow including data preprocessing, exploratory analysis, feature engineering, model building, and performance evaluation.

2. 🎯 Problem Statement

Banks face significant financial risk due to loan defaults.
This project aims to:
Analyze customer financial behavior
Identify high-risk loan applicants
Build a predictive model to classify loan default probability
Improve decision-making in loan approvals

3. 📊 Exploratory Data Analysis (EDA)

Key analysis performed:
Loan distribution across income groups
Credit history impact on loan approval
Relationship between loan amount and default rate
Correlation analysis of financial attributes
